# 📊 Controle de Caixa 

Projeto em Java que simula um controle de caixa simples, permitindo registrar entradas e saídas,
calcular saldo final e gerar um relatório básico de movimentações.

## 🧠 Problema

Pequenos negócios ou controles pessoais muitas vezes precisam registrar entradas e saídas
de dinheiro de forma simples, sem depender de planilhas ou sistemas complexos.

## ✅ Funcionalidades

- Abertura e fechamento de caixa
- Registro de movimentações de ENTRADA e SAÍDA
- Cálculo automático do saldo
- Relatório final do caixa
- Validações de dados (valores, tipos e estado do caixa)

## 🧱 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Construtores
- Atributos de classe (static)
- Validações de regras de negócio
- Manipulação de arrays

---

### Observações
- **Projeto desenvolvido com foco em aprendizado e prática de conceitos fundamentais de Java e POO.
Não utiliza frameworks ou bibliotecas externas.**
